## Description

**Short (280 chars):** Run the IT interview gauntlet of 2026. Dodge red flags, ship commits, ace 8 roles. 5 QTE mechanics, 40+ scenarios, Wordle-style share card. Rec2it.run.

**Medium (500 chars):** Every IT job seeker knows: 200 candidates per role, AI filters, fake offers. Job Interview Runner 2026 turns that pain into a 2-minute game. Sprint through a corporate hallway, dodge toxic interview tropes (red flags, impostor syndrome), and ace timed challenges in 8 different roles — Junior Frontend, Senior Fullstack, DevOps, ML, PM, QA, Mobile. Each role has unique QTE scenarios: spot the SQL injection, sequence the system design layers, hold for a long compile. Share your score on Twitter with a Wordle-style emoji grid.

**Long:** *For press releases, blog posts.*

Job Interview Runner 2026 is an indie web game that satirizes the modern IT hiring process. Built with PixiJS v8 and React 18, the game delivers a fast-paced endless runner with quick-time events that mirror real interview scenarios — from CSS bugs to k8s deployments. Players choose from 8 roles, each with unique stats and QTE templates derived from actual industry scenarios.

The game features:
- **40+ QTE scenarios** based on real IT interview problems
- **8 roles** with distinct gameplay (Junior Frontend → Mobile Developer)
- **Roguelike modifiers** (overtime_expected, toxic_pm, layoff_season) that change each run
- **Wordle-style share cards** with emoji grid — proven viral mechanic
- **Supabase leaderboard** with server-side HMAC anti-cheat
- **PWA-ready** — installable to home screen, offline-capable
- **Mobile-optimized landing page** for cross-platform sharing

## Key Differentiators

1. **Niche-specific content** — every QTE references real IT scenarios (SQL injection, race conditions, system design). No generic "click fast" filler.
2. **Viral by design** — Wordle-style share card with 6 emoji grid + score. Tweeted within seconds of game over.
3. **No paywall in core loop** — sponsor banners only at EndScene; never blocks gameplay.
4. **Privacy-first** — Plausible analytics (no cookies), opt-in leaderboard (no email), GDPR-compliant.
5. **Russian + English** — original content in Russian (where the joke lands hardest), translated to English for global reach.
