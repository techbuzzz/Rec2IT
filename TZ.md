# Job Interview Runner — Техническое Задание

> **Концепция:** браузерная инди-игра (endless runner) про IT-собеседования в 2026.
> **Хайп-механик:** шаринг скоринговой карточки в Twitter/Reddit/LinkedIn → вход в глобальный leaderboard.
> **Эмоциональный якорь:** «200 кандидатов на место, пройди собесы быстрее всех и получи оффер».

---

## 1. High-Level Описание

**Жанр:** Endless runner (2D, side-scroll, top-down или 3/4 view) + QTE-вставки на собеседованиях.

**Игрок:** выбирает роль при старте (HR, Backend Dev, Frontend Dev, QA, DevOps, Data Scientist, PM, Designer) — у каждой свой стартовый скилл-сет и свой набор мемных собеседований.

**Геймплей:** бежишь по «карьерной лестнице», уворачиваясь от багов, дедлайнов, выгорания и неадекватных задач. Через каждые N секунд (или метров) — экран собеседования: отвечаешь на вопросы, выбираешь ответы, проходишь QTE на кодинг-челлендже.

**Цель:** пройти все уровни собесов и получить offer letter. Финальный скоринг = время + правильные ответы + бонусы за стрик без ошибок.

**Шаринг:** на финише — карточка-скрин (как Wordle): «Я получил оффер в Google за 3:47, обойди меня» → share в Twitter/Reddit/LinkedIn → ссылка ведёт на игру + показывает глобальный leaderboard.

---

## 2. Целевая Аудитория и Почему Зайдёт

- **IT-специалисты 25-45 лет** — все проходили этот ад, боль универсальная
- **Рынок 2026:** 200 кандидатов на место, массовые сокращения, ИИ заменяет junior'ов, требования растут
- **Психологический посыл:** «спустить пары» через мем и black humor, не через жалобы
- **Социальная мотивация:** каждый хочет доказать, что он быстрее/умнее других

---

## 3. Игровые Роли (8 стартовых)

| Роль | Стартовый бонус | Мемная фраза при старте |
|------|----------------|------------------------|
| HR | Видит подсказки в диалогах | «У меня 500 откликов, и я сама их читаю» |
| Backend Dev | +30% к скорости | «А у нас микросервисы на 500 сервисов» |
| Frontend Dev | Удвоенные бонусы за UI-квесты | «Pixel-perfect? В 2026-м?» |
| QA | Иммунитет к багам на первом уровне | «А я баг нашла в твоём резюме» |
| DevOps | Регенерация выносливости | «K8s упал, но не я» |
| Data Scientist | Бонус к Data-квестам | «Это не bias, это фича» |
| PM | Видит roadmap уровней заранее | «Спринт на 2 недели, ща будет» |
| Designer | Бонус к визуальным QTE | «Я не буду это пиксель-пиксель править» |

## 4. Уровни Собеседований (5 этапов, нарастающая сложность)

### Уровень 1: HR-Screen
- **Вопросы:** «Почему вы ушли с прошлой работы?» (5 вариантов, 1 правильный — не токсичный)
- **QTE:** листать резюме (стрелки ←/→), найди опечатку за 3 сек
- **Мем:** «Мы вам перезвоним» (никогда не перезвонят)

### Уровень 2: Технический скрининг
- **Вопросы:** базовые по стеку роли (множественный выбор)
- **QTE:** быстро напечатать `git commit -m "fix"` (опечатки = -score)
- **Мем:** «А почему в резюме 14 мест работы за 10 лет?»

### Уровень 3: Live Coding
- **QTE:** мини-LeetCode задача (Easy уровень, 60 сек)
  - Написать функцию reverse string
  - Найти дубль в массиве
  - FizzBuzz
- **Мем:** «У нас whiteboard-интервью, маркер есть свой»

### Уровень 4: System Design
- **Вопросы:** «Как бы вы спроектировали Twitter?» (варианты компонентов)
- **QTE:** расставить кубики (сервисы) в правильном порядке
- **Мем:** «А давайте начнём с того, как работает интернет»

### Уровень 5: Final + Offer
- **Культурный fit:** «Вы готовы к переработкам?»
- **QTE:** подписать оффер (mouse-only, быстро)
- **Финал:** анимация получения оффера, генерация карточки

---

## 5. Беговые Механики (между уровнями)

**Препятствия (уклонения):**
- 🐛 Баг (нужно прыгнуть)
- 📅 Дедлайн (нужно присесть)
- 💼 Токсичный менеджер (нужно отпрыгнуть в сторону)
- 🔥 Выгорание (медленно ползёт, нельзя касаться)
- 🤖 ИИ-рекрутер (стреляет авто-отказами)

**Бонусы (собираемые):**
- ☕ Кофе (+скорость на 3 сек)
- 💰 Зарплата (мелочь, +1 очко)
- 🏆 Рекомендация (+иммунитет)
- 📚 Курс на Coursera (мемный — рандомно -5 очков)
- 🎯 Реферал (×2 очки на следующем уровне)

**Расходники:**
- ❤️ Выносливость (3 жизни, теряются от багов/выгорания)
- ⚡ Концентрация (нужна для QTE, уменьшается со временем)

**Управление:**
- ↑/Space — прыжок
- ↓ — присед
- ←/→ — переключение полосы
- Любая клавиша / клик — QTE

## 6. Скоринг и Финальная Карточка

**Формула скоринга:**
```
Score = (time_bonus) + (correct_answers × 100) + (streak_bonus) + (role_bonus) - (mistakes × 50)
```

**Карточка шаринга (генерируется как PNG на canvas):**

## 7. Глобальный Leaderboard

**Бэкенд (Supabase или Firebase free tier):**

**Таблица `scores`:**
- `id` (uuid)
- `player_name` (text, вводится при шаринге, опционально)
- `role` (text)
- `time_seconds` (int)
- `score` (int)
- `created_at` (timestamp)
- `country` (text, по IP через Cloudflare)

**Views:**
- Top-100 за всё время
- Top-100 за неделю
- Top-100 по каждой роли
- «Personal best» с session-id в localStorage

**Anti-cheat:**
- Скоринг подписывается HMAC на клиенте
- Аномальные значения (score > 5000 или time < 60 сек) → флаг модерации

## 8. Технический Скоуп

**MVP (что нужно для запуска за 2-3 недели):**

### Frontend
- **Стек:** Vanilla JS + HTML5 Canvas (для скорости и простоты деплоя)
- **Альтернатива:** React + Phaser 3 (если хотим красивее и быстрее в разработке геймплея)
- **Сборка:** Vite, single-page app
- **Деплой:** Vercel / Netlify / GitHub Pages

### Backend
- **Supabase** (Postgres + Auth + Realtime) — leaderboard
- **Cloudflare Workers** — генерация OG-image для шаринга (опционально)
- **Plausible / Umami** — простая аналитика

### Ассеты
- **Спрайты:** Kenney CC0 pack + свои пиксельные (8-bit/16-bit стиль)
- **Звук:** freesound.org CC0 — пиканье, успех, fail
- **Шрифт:** Press Start 2P (Google Fonts) для олдскульного feel

### Тестирование
- Jest / Vitest для логики скоринга
- Cypress / Playwright для E2E (1 happy path)
- Manual QA на 5 устройствах

## 12. Риски и Митигация

| Риск | Вероятность | Митигация |
|------|-------------|-----------|
| Забанит Twitter за «спам-шаринг» | Средняя | Делать web intent (не автопубликация), оставлять на пользователя |
| Забанит Reddit за низкокачественный пост | Средняя | Лучший контент — сам геймплей, в посте — видео + ссылка |
| Игроки заспидранят и сломают скоринг | Высокая | HMAC + серверная валидация + rate limit |
| Не взлетит в первый месяц | Высокая | Бюджет на буст в Hacker News / Product Hunt как fallback |
| Утечка кода / DDoS | Низкая | Cloudflare proxy, Supabase RLS |
